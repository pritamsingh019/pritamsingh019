## What I'm building

I study AI & Cybersecurity at IIT Patna — but my real question is simpler: why does understanding your own data require hiring someone with a PhD?

**DataScout** is my answer. An LLM agent that compiles natural language into sandboxed Python, deployed serverless on AWS Lambda + S3, cutting analysis from hours to seconds.

I didn't just build the AI — I built the **secure execution cage** around it. Because an agent that runs arbitrary code and one that does so safely are two very different things.

Now learning **Rust** — because fast models need fast infrastructure, and LLMs at the edge need systems programmers.

## Principles

- **If it needs a framework, it's not understood yet** — build from scratch until every line is explainable
- **Safety is not a feature** — a sandboxed agent and an unsandboxed one are not the same product
- **Real deployment > perfect paper** — DataScout runs on AWS. Drip_sense notifies real nurses. Code that solves real problems.
- **AI that gatekeeps intelligence has failed** — if a non-expert can't use it in 5 minutes, I'm not done building
